安装并不复杂 next -> next -> next…
检测安装成功：

  • 打开命令行 Windows + R， 打开CMD

所有命令，都在这里输入

  • 输入命令， 查看当前 node 版本。可查到版本号，nodejs安装成功 

      npm -version

1 借力

  • 利用 npm， 安装cnpm， 借用淘宝镜像.

    稍微等待。
    安装成功，下一步。
    安装失败，重来。

    npm install -g cnpm –registry=http://registry.npm.taobao.org
    查看安装版本

    cnpm -v

2 框架

  • 安装 hexo 框架

稍微等待。
安装成功，下一步。
安装失败，重来。

cnpm install -g hexo-cli

查看安装版本 

hexo -v

3 地基

  • 新建博客文件

查看当前目录

chdir

C：\ Users\ yourname

在该目录下，新建文件夹 blog。
此后，博客相关的文件，都会保存在 blog 文件夹里。
如果后续步骤出错，难以解决，可直接删除 blog 文件夹， 从此步骤重新开始。

mkdir blog
cd blog

初始化博客。
hexo 为你的博客，自动 clone 初始文件。 

hexo init

等待。 祈祷无error

Start blogging with Hexo!

初始化成功

hexo server

Hexo is ruuning at http： // localhost:4000 . Press Ctrl + C to stop

打开地址

选好了工具，看看怎么用 markdown

5 部署

  • 把博客部署到Github！

（1）在 github 上 create a new repository 。 命名仓库，要这样(yourname 是 你的 github 账号名！）：

yourname.github.io

（2） 打开cmd， 命令行：

cnpm install --save hexo-deployer-git

（3） 打开 blog 文件夹，找到 _config.yml , 打开编辑（VS Code ！）

_config.yml 找到 #Deployment ，并添加以下内容 

repo：https://github.com/yourname/yourname.github.io.git       // 新建的仓库地址 
branch： master
type： 'git'
message: blog

修改完成，保存。回到命令行：

hexo d

按照提示，输入 github 账号密码
成功后，刷新 github 仓库，可见本地 blog 文件夹在仓库里！
在本地编辑好博文后， 依次输入以下命令行，即可更新博客！

hexo clean
hexo g      
hexo d

done，通过博客地址可看到更新：https://yourname.github.io/

6 主题

  • hexo 框架初始化主题为 landscape， 修改流程为：
    clone 主题 -> 修改 _config.yml -> 更新

clone 

git clone https://github.com/theme-next/hexo-theme-next themes/next             //next 主题

blog 文件夹， 打开 _config.yml
找到 theme：
修改要使用的主题名为 next 

theme: next

update: 

hexo clean
hexo g
hexo d

done，通过博客地址可看到更新：https://yourname.github.io/
