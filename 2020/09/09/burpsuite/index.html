burpsuite 曲折破解记

在写攻防世界Web_weak_auth，
问题在于：破解账户名和密码

“login as admin” 提示了用户名为 admin，
题目描述的孩子，呆呆地，“随手设置了密码”
这份呆呆，容易联想到弱口令，
反手一试123456，flag正面来迎

但复盘再看大佬的Write up， 不少用的暴力求解
有python写脚本，有burpsuite抓包攻击
秉持着“亲自多试试”，“拓展思路”的想法
我也要试试爆破密码!

工欲善其事必先利其器

（当时并没意识到：工具准备两小时，解题夺旗十分钟）

这个工具，burpsuite的安装配备，

走了太多多多多弯路
为避免再次踩坑、规避非技术的费心、整合安装资源，记录这篇曲折blog

burpsuite 可在官网上下载， professional 要收费，399刀一年。
我偏要破解，**走弯路的时间成本或许抵得上官网的收费吧。**
以下将介绍通过burp-loader-keygen，来破解burpsuite。

0 环境配备

  • 运行burpsuite，要安装java运行环境，推荐
    直接点击下载 JAVA

安装时，点击点击点击下一步，
系统默认创建了新路径C:\Program Files\Java，存储JDK 和 JRE 文件。

1 破解工具

  • 下载burpsuite及burp-loader，推荐
    直接点击下载破解工具

推荐下载的java和burpsuite，版本都相适应使用。
如版本不适配，影响安装。

坑_a

在第一遍安装java环境时，并没完全成功，一开始没意识到，
到破解时点击“run”丝毫没有响应，才怀疑可能是环境的问题。
当时的检查：在CMD 中敲 java ，出现java指令的用法，
但敲javac，出现：*’javac’ 不是内部或外部命令，也不是可运行的程序或批处理文件。*
google 后，知道要添加环境变量，按着前人的脚步搞好了。

2 破解过程

  • 配置好java环境，可选择java打开方式burp-loader-keygen-2_1_06.jar，
    （正常来说，双击即可打开）
    点击run，会打开burpsuite界面

  • 复制license中的一大串东西，粘贴到burpsuite界面
    点击选择手动激活（就是点击manual ××× ），
    然后点击next按钮
    再复制burpsuite页面下边出现的一大串东西，
    粘贴到license第二个框框
    （这些破解步骤easy，Google it when faced difficulties)

坑_b

填了第一个坑，环境变量弄好，
CMD中java、javac指令都有响应了
然而破解时点击“run”还是打不开
就试着用CMD打开 ： 

javaw -jar burp-loader-keygen-2_1_06.jar

“run” 就有响应了！

3 快捷方式

每次要打开burpsuite，总不能复制来复制去的
就在路径 C:\Program Files\Java 中新建了burp.bat
输入 

javaw -jar burp-loader-keygen-2_1_06.jar

保存后，发送到桌面快捷方式。
